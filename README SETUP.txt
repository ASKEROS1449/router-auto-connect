Папки ОБЯЗАТЕЛЬНО нужно скопировать на локальную машину для корректной работы. С \\192.168.1.73\ работать НЕ БУДЕТ!!!

Chrome:
Открыть chrome://extensions/, в верxнем левом углу окна выбрать "Load Unpacked", выбрать папку "Opera + Chrome", расширение сразу отобразится первым в списке.

Opera:
Открыть extensions/, в правом верxнем углу окна включить "Developer Mode", выбрать "Load Unpacked Extension", выбрать папку "Opera + Chrome", расширение сразу отобразится первым в списке.

Mozilla:

Открыть about:debugging#/runtime/this-firefox, выбрать "Load Temporary Add-on...", выбрать файл manifest.json из папки "Firefox".

По поводу проблем обращаться IP 791.
