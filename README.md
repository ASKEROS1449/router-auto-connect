# Router Auto-Connect Extension

Расширение для автоматического поиска и открытия веб-интерфейсов оборудования на служебных IP-адресах.

## Возможности
- Автоматически определяет попытку входа на IP из диапазонов `100.60.x.x` и `5.197.x.x`.
- Сканирует порты **443, 8080, 8888** и **80**.
- Автоматически обходит ошибку SSL (Self-Signed Certificates) для мгновенного доступа.
- Приоритет отдается рабочим HTTPS интерфейсам.

## Установка

### Google Chrome / Opera / Edge
1. Скачайте код или клонируйте репозиторий.
2. Откройте `chrome://extensions/`.
3. Включите **Режим разработчика** (справа сверху).
4. Нажмите **Загрузить распакованное расширение** и выберите папку `chrome`.

### Mozilla Firefox
1. Откройте `about:debugging#/runtime/this-firefox`.
2. Нажмите **Загрузить временное дополнение**.
3. Выберите файл `manifest.json` из папки `firefox`.

## Безопасность
Расширение использует `Debugger API` для автоматического пропуска предупреждений сертификатов безопасности на целевых IP.
